# ポケコン PC-G850用ゲーム 『バナナ・モール　BANANA MOLE』

## 動作機種
SHARP PC-G850系

## 開発言語
ポケコン本体又はエミュレータ（以下本体と記す）内蔵のＣ言語<br>

## 同梱ファイル

| ファイル名  | 内容                                       |
|-------------|--------------------------------------------|
| `README.md` | マニュアルファイル                         |
| `extlib.h`  | Ｃ言語ソースファイル（拡張関数ライブラリ） |
| `stage.h`   | Ｃ言語ソースファイル（ステージデータ）     |
| `mole.c`    | Ｃ言語ソースファイル（メイン）             |

## セットアップ

本体へ、以下の順でセットアップします。<br>

### PC-Gリンク以外の転送ソフトをお使いの場合

ソースリストに行番号が付いていませんので、ポケコンのSIO設定でLine Numberを"`no`"に変更してください。<br>

* `extlib.h`を転送し、同名でファイルを保存します。
* `stage.h`を転送し、同名でファイルを保存します。
* `mole.c`を転送します。（※ファイルは保存しません）

### PC-Gリンクをお使いの場合

ソースリストをPC-Gリンクのウインドウで開いた時点で行番号が付きますので、ポケコンのSIO設定のLine Numberを変更する必要はありません。<br>

* `extlib.h`のファイル名を"`extlib.c`"に変更して転送し、"`extlib.h`"の名前でファイルを保存します。
* `stage.h`のファイル名を"`stage.c`"に変更して転送し、"`stage.h`"の名前でファイルを保存します。
* `mole.c`を転送します。（※ファイルは保存しません）

※本体のRAM容量が標準（未改造・32KB）の場合、出来るだけ本体の容量を空けておいてください。（理由は後述）<br>

## ゲームのジャンル

パズルゲーム<br>

## 起動方法

本体の`SHIFT(2ndF)`+`TEXT`キーを押下し、本体内蔵のＣ言語を起動し、`Go`（`Gキー`を押下）でコンパイル及び起動します。<br>

## 操作方法

### 《タイトル画面》

| キー        | 役割                                                                                              |
|-------------|---------------------------------------------------------------------------------------------------|
| `8・2キー`  | ステージセレクト（1～20）（8:加算・2:減算）                                                       |
| `+・-キー`  | ゲームの速度調整（1～99）（+:速く・-:遅く）                                                       |
| `Sキー`     | 音のOFF/ONの切り替え<br> ※本体に圧電ブザーが取り付けられていない場合、ONにしても音は鳴りません。 |
| `SPACEキー` | ゲームスタート                                                                                    |

### 《ゲーム》

| キー                       | 役割                               |
|----------------------------|------------------------------------|
| `８`・`２`・`４`・`６キー` | 上下左右へ移動                     |
| `SPACEキー`                | ギブアップ、各種メッセージスキップ |

## 遊び方

モール君を操作し、全ての食べ物やお宝（さつまいも・バナナ・苺・アイスクリーム・ドル袋・金塊）を取り、家まで帰ってください。<br>

モール君は、空白の移動はもちろん、モグラなので土を掘って移動することができます。<br>
また、ハシゴを昇降することもできます。<br>
しかし、モール君の下に穴（空白のところ）があると、空白以外のところまで落下してしまいます。<br>

岩は、下に土が無くなると、落下してしまいます。<br>

なお、モール君は鍛えているので、頭上から岩が落下しても耐えられますが、通り抜けることはできません。<br>

モール君が移動できなくなった場合や解けなくなった場合は、ギブアップでやり直ししてください。<br>

ギブアップを３回すると、ゲームオーバーとなります。<br>

全部で２０ステージまで用意してあります。<br>
２０ステージ目をクリアすると、エンディングとなります。<br>

## 終了方法

本体の`ON(BREAK)キー`や`OFFキー`を押してください。<br>

## うまくコンパイル・動作しない場合

このゲームでは、本体内蔵のＣ言語を使用している都合上、本体の残りRAM容量は充分余裕を取る必要があります。<br>

コンパイルが通らない場合や、ゲーム動作中に異常終了してしまう場合、このゲーム以外のファイルを削除してください。<br>
また、機械語のユーザーエリアも確保しないでください。<br>
（機械語モニタから「`USER 00FF`」を実行）<br>

## ステージコンストラクション

　ステージデータファイル（`stage.h`）を変更することにより、自由にステージ内容を編集できます。<br>

　ファイルの５行目以降から、各６行分が１ステージとなります。<br>
　（縦６×横１８のサイズで、画面のイメージ通りとなります。）<br>

　ダブルクォーテーション（`""`）内の英数字は、以下の通りです。<br>

| 番号 | 種類           |
|------|----------------|
| 0    | 空白           |
| 1    | 土             |
| 2    | 岩             |
| 3    | ハシゴ         |
| 4    | 家             |
| 5    | さつまいも     |
| 6    | バナナ         |
| 7    | 苺             |
| 8    | アイスクリーム |
| 9    | ドル袋         |
| A    | 金塊           |
